Pengenalan Python

Python adalah bahasa pemrograman yang mudah dipelajari dan digunakan untuk berbagai tujuan.

Variabel dan Tipe Data


nama = "Budi" umur = 25 is_student = True print(nama) print(umur) print(is_student)

Variabel digunakan untuk menyimpan data. Python memiliki berbagai tipe data, seperti string, integer, dan boolean.

Struktur Percabangan


umur = 18 if umur >= 18: print("Dewasa") else: print("Anak-anak")

Percabangan digunakan untuk pengambilan keputusan berdasarkan kondisi yang diberikan.

Perulangan


for i in range(5): print(i)

Perulangan digunakan untuk menjalankan suatu blok kode secara berulang.

Fungsi


def sapa(nama): print("Halo, " + nama)

sapa("Budi")

Fungsi digunakan untuk mengelompokkan kode yang dapat dipanggil berulang kali.

Dictionary


data = {"nama": "Budi", "usia": 25} print(data["nama"])

Dictionary adalah struktur data yang menyimpan data dalam format key-value.

List


data = [1, 2, 3, 4] print(data[0])

List adalah struktur data yang dapat menyimpan beberapa nilai dalam satu variabel.

Set


data = {1, 2, 3, 3, 2} print(data)

Set adalah kumpulan elemen unik tanpa urutan tertentu.
